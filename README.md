# pythonica
 A very simple Python GUI interface made with pygame to communicate with a chosen ollama model

## How to run 

-Install astral uv Python package manager
-Clone this repository 
-Ensure that your model is already running (ollama ps, otherwise - ollama run yourmodel_name)
-Change the model name in the main.py if you do not use gemma3:270m
-uv run main.py

Have a nice chat with your model
